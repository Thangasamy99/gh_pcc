# PCC Health Services - Standardized Report Header

To ensure professional branding across all HMS printouts, follow this structure for document headers.

## 1. Official Branding Hierarchy
- **Primary Title:** PRESBYTERIAN CHURCH IN CAMEROON (Uppercase, Bold)
- **Department:** HEALTH SERVICES (Uppercase)
- **Motto:** Compassionate Quality Healthcare (Italicized or Subtext)

## 2. Branding Elements
- **Logo Left:** PCC Standard Circular Cross Logo
- **Logo Right:** PCC Health Services Logo (Rod of Asclepius)
- **Contact Info:** Synod Office, P.O. Box 19 Buea, South West Region, Republic of Cameroon.
- **Tel:** +237 233322487 / +237 677579487
- **e-mail:** moderator_office@pcconline.org

## 3. Visual Layout
The header should be centered between the two logos, with a bold horizontal line separating it from the content.

## 4. Usage Guidelines
- **Dashboard Headers:** Use yellow banner style branding.
- **Official Reports (Invoices, Lab Results, Receipts):** Use the circular seal and formal letterhead layout.
